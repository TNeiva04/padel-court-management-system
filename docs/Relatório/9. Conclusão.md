# Conclusão

O desenvolvimento deste projecto permitiu criar uma aplicação web funcional e interactiva, a partir de um backend simples. Ao longo do processo, fomos implementando várias melhorias importantes, como a gestão completa de rentals com suporte para operações PUT e DELETE, autenticação segura através de tokens, e uma interface mais rápida  graças à arquitectura Single Page Application (SPA), no qual estas melhorias ajudaram a tornar a aplicação mais fácil de usar, com respostas mais rápidas e uma navegação mais simples entre as várias partes do site.

Para além disso, aprendemos a organizar melhor o código, a modularizar o frontend e a optimizar as consultas à base de dados para dar resposta a novos casos de uso. Por fim, realizámos o deploy da aplicação com sucesso, utilizando Docker e a plataforma Render, o que nos permitiu consolidar conhecimentos essenciais e ganhar prática no desenvolvimento do projecto.
